# 문제

정수들이 하나의 벡터에 순서 없이 주어진다.  
우리는 “체력”을 나타내는 양의 정수 `x`를 정하고,  
벡터의 원소들을 **어떤 순서로든** 골라서 `흡수`하는 과정을 진행한다.

## 흡수 규칙

- 시작 체력: `cur = x` (항상 `cur ≥ 1`이어야 한다).
- 아직 사용하지 않은 원소 중에서 하나를 골라 값 `v`를 선택한다.
- 이 값을 흡수하려면 다음 조건을 만족해야 한다.

  - `cur ≥ |v| + 1`

- 흡수에 성공하면 

  - `cur ← cur + v`

- 흡수 도중 언제라도 `cur`가 1 미만(≤ 0)이 되면 안 된다.
- 각 원소는 **최대 한 번만** 흡수할 수 있다.

## 목표

벡터의 길이가 `n`일 때,  
각 `t = 1, 2, ..., n`에 대해 다음을 정의한다.

- `x_t` = 정확히 `t`개의 원소를 어떤 순서로 흡수할 수 있는  
  **가장 작은 시작 체력 x (정수, x ≥ 1)**

여기서 “흡수할 수 있다”는 말은,  
어떤 순서와 선택을 통해 위 흡수 규칙을 만족하면서  
정확히 `t`개의 서로 다른 원소를 흡수하는 시나리오가 존재한다는 뜻이다.

당신의 작업은,  
주어진 벡터에 대해 `[x_1, x_2, ..., x_n]` 을 구하는 것이다.  
만약 어떤 `t`에 대해 어떤 시작 체력으로도 `t`개를 흡수하는 것이 불가능하다면  
`x_t = -1`로 둔다.

---

## 입력

- 첫 줄에 정수 `n`이 주어진다.  
- 둘째 줄에 정수 `n`개 `a_1, a_2, ..., a_n` 이 주어진다.

제한:

- `1 ≤ n ≤ 200000`
- `-10^9 ≤ a_i ≤ 10^9`

---

## 출력

- `n`개의 정수 `x_1, x_2, ..., x_n`을 공백으로 구분해 한 줄에 출력한다.
- 각 `x_t`는 정확히 `t`개의 원소를 흡수할 수 있는 최소 시작 체력이다.
- 불가능한 경우에는 해당 위치에 `-1`을 출력한다.

## 답 (code)

아래 코드는 “논리적으로 일관된 해답”으로,  
각 \(k\)에 대해 이분 탐색으로 \(x_k\)를 구하는 방식이다.  
(시간 최적화는 별도 문제로 두고, 문제 정의를 만족하는 구현에 집중했다.)

```cpp
#include <bits/stdc++.h>
using namespace std;
using int64 = long long;
const int64 INF = (int64)4e18;

// 정렬된 배열 a와 시작값 start가 주어졌을 때,
// 주어진 규칙으로 "최대로" 몇 개까지 흡수할 수 있는지 계산한다.
int maxAbsorbCount(const vector<int64>& a, int64 start) {
    int n = (int)a.size();
    int64 cur = start;
    int cnt = 0;

    for (int i = 0; i < n; ++i) {
        int64 v = a[i];
        // cur가 v 이상이면 흡수
        if (cur >= v) {
            cur += v;
            if (cur < 0) {
                // cur가 음수가 되면 더 진행할 수 없으므로 중단
                return cnt;
            }
            cnt++;
        }
        // cur < v 이면 이 원소는 사용할 수 없고 그냥 지나간다
    }
    return cnt;
}

// 정렬된 배열 a 에 대해,
// 정확히 k개를 흡수할 수 있는 최소 시작값 x_k 를 이분 탐색으로 구한다.
// 불가능하면 -1을 반환한다.
int64 minStartForK(const vector<int64>& a, int k) {
    int n = (int)a.size();

    // 우선, "아무리 큰 시작값을 줘도 k개를 못 먹는 경우"를 판정하기 위해
    // 충분히 큰 시작값에서 시도해 본다.
    int64 testStart = (int64)1e12;
    if (maxAbsorbCount(a, testStart) < k) {
        return -1; // k개 흡수가 불가능
    }

    int64 lo = 0, hi = testStart;
    int64 ans = testStart;

    while (lo <= hi) {
        int64 mid = (lo + hi) / 2;
        int cnt = maxAbsorbCount(a, mid);

        if (cnt >= k) {
            // mid로도 k개 이상 흡수 가능하므로, 더 작은 시작값이 있는지 탐색
            ans = mid;
            hi = mid - 1;
        } else {
            // mid로는 k개를 못 먹는다 → 더 큰 시작값이 필요
            lo = mid + 1;
        }
    }
    return ans;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int n;
    cin >> n;
    vector<int64> a(n);
    for (int i = 0; i < n; ++i) cin >> a[i];

    // 순서가 중요하지 않으므로 오름차순 정렬
    sort(a.begin(), a.end());

    vector<int64> answer(n);
    for (int k = 1; k <= n; ++k) {
        answer[k-1] = minStartForK(a, k);
    }

    for (int i = 0; i < n; ++i) {
        cout << answer[i] << (i + 1 == n ? '\n' : ' ');
    }
    return 0;
}
```
