# 문제 설명

격자 위에서 일부 칸을 반드시 방문해야 할 때, 최소 이동 비용을 구하는 문제이다.

격자의 크기 w × h 가 주어진다.
격자의 각 칸에는 왼쪽 위부터 오른쪽으로, 위에서 아래로 번호가 매겨진다.

- 행 번호는 위에서 아래로 0부터 h-1
- 열 번호는 왼쪽에서 오른쪽으로 0부터 w-1
- 칸 번호는 다음과 같이 정의된다.
  번호 = y * w + x + 1

예를 들어, w = 3, h = 2 일 때 칸 번호는 다음과 같다.

1 2 3  
4 5 6

상하좌우로 인접한 칸으로만 이동할 수 있으며,
인접한 칸으로 1번 이동할 때마다 비용은 1이다.

방문해야 하는 칸들의 번호가 주어진다.
이 중 번호 1은 항상 포함되며, 출발 위치는 반드시 칸 1이다.

- 출발 위치: 칸 1
- 모든 “방문해야 하는 칸”을 한 번 이상 방문해야 한다.
- 출발한 칸 1로 다시 돌아올 필요는 없으며, 어느 칸에서 끝나도 된다.
- 목표: 모든 방문해야 하는 칸을 방문하는 경로 중, 총 이동 비용의 최솟값을 구하라.

---

## 입력 형식

- 첫 줄: 정수 w, h — 격자의 가로, 세로 크기
- 둘째 줄: 정수 k — 방문해야 하는 칸의 개수
- 셋째 줄: 정수 k개 — 방문해야 하는 칸의 번호들 (공백으로 구분)

조건:

- 1 ≤ w, h
- 1 ≤ k ≤ w * h
- 모든 칸 번호는 1 이상 w * h 이하의 서로 다른 정수
- 방문해야 하는 칸 번호들 중 항상 1이 포함된다.

---

## 출력 형식

모든 방문해야 하는 칸을 한 번 이상 방문하는 경로들 중에서,
총 이동 비용(이동한 칸 수)의 최솟값을 정수 하나로 출력한다.

---

## 예시

### 입력 예시

w = 3, h = 2, k = 3  
방문해야 하는 칸 번호: 1 3 6

격자는 다음과 같다.

1 2 3  
4 5 6

가능한 경로 중 하나는 1 → 3 → 6 이다.

- 1에서 3까지: 2칸 이동
- 3에서 6까지: 1칸 이동

총 이동 비용은 3이며, 이 경우가 최솟값이 된다.
따라서 출력은 3 이다.

## 답 (code)

```cpp
#include <bits/stdc++.h>
using namespace std;

const int INF = 1e9;

// 번호 n(1-base) -> (x, y) 좌표
pair<int,int> cellToCoord(int n, int w) {
    int idx = n - 1;
    int y = idx / w;
    int x = idx % w;
    return {x, y};
}

// 방문해야 하는 칸들의 좌표 벡터를 만든다.
vector<pair<int,int>> buildPoints(const vector<int>& cells, int w) {
    int k = (int)cells.size();
    vector<pair<int,int>> pts(k);
    for (int i = 0; i < k; ++i) {
        pts[i] = cellToCoord(cells[i], w);
    }
    return pts;
}

// pts 사이의 맨해튼 거리로 dist[i][j] 완전 그래프를 만든다.
vector<vector<int>> buildDist(const vector<pair<int,int>>& pts) {
    int K = (int)pts.size();
    vector<vector<int>> dist(K, vector<int>(K, 0));
    for (int i = 0; i < K; ++i) {
        for (int j = 0; j < K; ++j) {
            int dx = abs(pts[i].first  - pts[j].first);
            int dy = abs(pts[i].second - pts[j].second);
            dist[i][j] = dx + dy;
        }
    }
    return dist;
}

// 시작 idx(startIdx)에서 시작해서,
// pts 전부를 방문하는 최소 비용 (시작점으로 돌아오지 않음)
int solveTSPPath(const vector<vector<int>>& dist, int startIdx) {
    int K = (int)dist.size();
    int FULL = (1 << K) - 1;

    static int dp[1 << 15][15];
    for (int m = 0; m <= FULL; ++m)
        for (int i = 0; i < K; ++i)
            dp[m][i] = INF;

    int startMask = (1 << startIdx);
    dp[startMask][startIdx] = 0;

    for (int mask = 0; mask <= FULL; ++mask) {
        for (int i = 0; i < K; ++i) {
            if (!(mask & (1 << i))) continue;
            int curCost = dp[mask][i];
            if (curCost == INF) continue;

            for (int j = 0; j < K; ++j) {
                if (mask & (1 << j)) continue;
                int nextMask = mask | (1 << j);
                int nextCost = curCost + dist[i][j];
                if (nextCost < dp[nextMask][j]) {
                    dp[nextMask][j] = nextCost;
                }
            }
        }
    }

    int ans = INF;
    for (int i = 0; i < K; ++i)
        ans = min(ans, dp[FULL][i]);
    return ans;
}

int main() {
    ios::sync_with_stdio(false);
    cin.tie(nullptr);

    int w, h;
    cin >> w >> h;

    int k;
    cin >> k;

    vector<int> cells(k);
    for (int i = 0; i < k; ++i) cin >> cells[i];

    // 방문해야 하는 칸들 좌표
    vector<pair<int,int>> pts = buildPoints(cells, w);

    // 시작 인덱스: 번호 1인 칸
    int startIdx = -1;
    for (int i = 0; i < k; ++i)
        if (cells[i] == 1) startIdx = i;

    if (startIdx == -1) {
        cout << 0 << '\n';
        return 0;
    }

    // 거리 행렬
    vector<vector<int>> dist = buildDist(pts);

    // TSP 변형 풀이
    int ans = solveTSPPath(dist, startIdx);
    cout << ans << '\n';
    return 0;
}
```

입력 예로 `w=3, h=2, k=3, cells = {1, 3, 6}`를 넣으면,  
경로 1 → 3 → 6의 비용 3이 정답으로 출력됩니다.
